# 🌐 WebScope

Website Intelligence Platform built using Python and FastAPI.

## Features

- Multi URL website analysis
- Parallel scraping
- Metadata extraction
- Internal link analysis
- AI-style insights
- Website scoring
- Dashboard interface
- CSV export

## Tech Stack

- Python
- FastAPI
- BeautifulSoup
- ThreadPoolExecutor
- HTML/CSS

## Run Locally

Install:

pip install fastapi uvicorn requests beautifulsoup4

Start:

uvicorn app:app --reload

Open:

http://127.0.0.1:8000
